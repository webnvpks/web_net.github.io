<div class="first">Text 2</div>
<div>Text 3</div>

jQuery:
$('#first').css ("color", "red");
var new = $("#first").clone ();
$("div.first:first").append (new)

В коде нет ошибок

В конце необходимо добавить точку с запятой

Нельзя создать переменную new, так как это зарезервированное имя
